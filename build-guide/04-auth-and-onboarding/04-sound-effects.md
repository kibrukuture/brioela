# 04 - Sound Effects

## Goal

Sound effects should make the memory world feel physical.

They must be subtle. If the SFX feel like a game trailer, they are too loud.

## Needed Sounds

```text
sfx-ember.wav
sfx-flame-bloom.wav
sfx-smoke-air.wav
sfx-water-shimmer.wav
sfx-cut-light.wav
sfx-stir-clay.wav
sfx-pot-resonance.wav
sfx-room-rise.wav
sfx-bite-burst.wav
sfx-particle-return.wav
sfx-brioela-note.wav
sfx-camera-open.wav
```

## Key Moments

- `flame` gets first physical bloom.
- `washed` gets water shimmer.
- `cut` gets a clean soft slice of light.
- `stirred` gets circular clay/wood movement.
- `inside a pot` gets low ceramic resonance.
- `through a bite` gets burst + whoosh + haptic.
- `Brioela begins` gets the strongest note.
- camera reveal gets a small lens/opening shimmer or no sound.

## Rule

Every sound must answer: what physical memory did this line awaken?
