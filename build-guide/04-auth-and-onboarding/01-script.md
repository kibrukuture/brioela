# 01 - Script

Status: current script is treated as final until we intentionally revise it.

## Final Script

```text
Before memory had words,
it had taste.

Before recipes,
there was flame.

Before maps,
there was smoke.

Before names,
there were hands.

Hands that washed.
Hands that cut.
Hands that stirred
until hunger became home.

A people could live
inside a pot.

A place could rise
from a smell.

A voice could return
through a bite.

Someone fed you.

And before you knew
what love was called,
your body knew
what love felt like.

One taste opens the door.
One breath brings back the room.
One bite returns the hands
that held you.

Brioela begins where language ends:
in the taste your body remembers
before your mind does.

Show Brioela something you eat.
```

## Script Job

The script should not explain Brioela. It should make the user feel the origin of Brioela.

It moves:

```text
memory before language -> fire -> hands -> culture -> place -> presence -> being fed -> Brioela -> camera
```

## Delivery Rules

- Slow enough to breathe.
- Not theatrical.
- Not soft in a weak way.
- Ancient, intimate, certain.
- Every pause is part of the script.
- Final spoken line should probably stop at `before your mind does.`
- `Show Brioela something you eat.` should probably appear as silent text over the camera.
