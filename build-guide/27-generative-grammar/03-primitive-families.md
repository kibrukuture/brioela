# Brioela Generative Grammar — Primitive Families

## What This File Covers

The primitive families that make the grammar expressive without relying on thousands of full components.

---

## Design Principle

Primitives are letters, not pages.

The goal is a small set of expressive building blocks that combine into many emotional UI moments while preserving Brioela's visual language.

## Primitive Count

Do not try to build 1000 components.

Target:

- 40-60 primitives
- 15-25 composition patterns
- 20-30 emotional/motion tokens

This is enough to create expressive combinations without losing maintainability.
