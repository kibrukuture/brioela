# 0001 — Chat Entrypoint

## Scope

Brain / Framework hardening

## What

Wire the `onMessage` / `chat()` entrypoint onto `BrioelaBrain`.

Receives a user turn, checks compression threshold (`checkCompressionNeeded`), runs compression if needed, calls the AI model with system prompt + memory tools + conversation history, appends the turn to `session_turns`, updates `inputTokens` + `turnCount` on the session, streams the response back.

This is the last piece — it connects every prior scope into a working agent.

## Verification

- `bun run verify`
- End-to-end: open session → send message → receive response → session_turns row written
- Compression fires correctly when thresholds exceeded
