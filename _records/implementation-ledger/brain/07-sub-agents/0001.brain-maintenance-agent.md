# 0001 — BrainMaintenanceAgent

## Scope

Brain / Sub-agents

## What

Create `BrainMaintenanceAgent` — ephemeral Durable Object keyed by `brain-maintenance-{userId}-{runId}`.

Fires on 7-day schedule via alarm. Before running, Brain checks user is idle (no active session in last 2h) — if not idle, reschedules 2h later. Agent reads user truth via typed Brain RPC (`readBrainContext`), reasons over memory patterns, writes conclusions back via typed Brain RPC (`writeBrainMemory`). Dies when work is done — no permanent state of its own.

Adds `brain_maintenance` alarm case to `dispatch.alarm.handler.ts`.

## Verification

- `bun run verify`
- Agent DO registered in wrangler config
- Alarm case dispatches to agent correctly
- All Brain writes go through typed RPC, not direct SQLite
