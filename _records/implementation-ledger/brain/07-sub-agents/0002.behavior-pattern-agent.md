# 0002 — BehaviorPatternAgent

## Scope

Brain / Sub-agents

## What

Create `BehaviorPatternAgent` — ephemeral Durable Object keyed by `behavior-pattern-{userId}-{runId}`.

Reads memory events via typed Brain RPC, detects behavioral patterns (food timing, recurring symptoms, dietary drift), writes detected patterns back as structured `user_memory` entries via typed Brain RPC. Dies when work is done.

Adds `behavior_pattern_detection` alarm case to `dispatch.alarm.handler.ts`.

## Verification

- `bun run verify`
- Agent DO registered in wrangler config
- Alarm case dispatches to agent correctly
- All Brain writes go through typed RPC
