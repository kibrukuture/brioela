# 0002 — System Prompt Builder

## Scope

Brain / Session lifecycle

## What

Create `build.system.prompt.handler.ts` — `buildSystemPrompt(db, sessionKind, userId)`.

Assembles blocks in strict prefix-cache order:
1. BrioelaIdentity (static constant — imported from `identity-prompt.ts`)
2. constraints (confirmed, always near top)
3. user_personality (active, ordered by strength DESC)
4. user_memory (relevant namespaces for this sessionKind)
5. skills index (name + description only)
6. previous session outcome_summary (most recent completed)

Static blocks before dynamic — deviation breaks Anthropic prefix cache.

## Verification

- `bun run verify`
- Block order is enforced — identity always first
- Returns a single joined string
