# 0003 — Session Close Handler

## Scope

Brain / Session lifecycle

## What

Create `close.session.handler.ts` — `closeSession(db, sessionId, outcomeSummary)`.

Sets session status to `completed`, writes `outcomeSummary`, sets `endedAt`. Cancels the pending session_watchdog alarm for this session. The `outcomeSummary` written here becomes Block 6 of the next session's system prompt.

## Verification

- `bun run verify`
- Session row status = `completed`, endedAt set
- Watchdog alarm row status = `cancelled`
