# 0001 — Session Open Handler

## Scope

Brain / Session lifecycle

## What

Create `open.session.handler.ts` — `openSession(db, userId, sessionKind)`.

Inserts a `sessions` row (status: active, inputTokens: 0, turnCount: 0), schedules a session_watchdog alarm (2h chat / 4h cooking), calls `buildSystemPrompt`, and returns `{ sessionId, systemPrompt }`.

Depends on: `build.system.prompt.handler.ts` (0002), `identity-prompt.ts` (agent-identity/0001).

## Verification

- `bun run verify`
- Session row written with correct fields
- Watchdog alarm row written to `scheduled_alarms`
