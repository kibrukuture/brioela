# 0007 — Web Tool

## Purpose

Implement the single AI-callable web tool: `search_web`.

The agent calls `search_web` when the user asks a question that requires current, factual, or external information that cannot be answered from brain memory or session context. It is the only tool in the tool set that makes an outbound network request.

The tool calls the Brave Search API (configured via `BRAVE_SEARCH_API_KEY` env variable). It returns a ranked list of web results: title, URL, and a snippet. The agent cites the URL in its response — it does not fabricate sources.

The tool is rate-limited to 5 calls per session to prevent abuse. A session-level counter is incremented on every call. When the limit is hit, the tool returns an error instead of calling the API.

`search_web` is only available in `general` sessions. Mira handles its own web lookups during cooking. Alarm sessions get no web access.

### 1. [NEW] [search.web.tool.ts](file:///Users/kibrukuture/Desktop/brioela/backend/src/agents/brain/_tools/search.web.tool.ts)

```typescript
import { tool } from 'ai'
import { z } from '@brioela/shared/zod'

const SESSION_WEB_SEARCH_LIMIT = 5

const searchWebSchema = z.object({
	query: z.string().min(1).max(300).describe('The search query. Specific and factual — not conversational.'),
	reason: z.string().min(1).describe('Why this web search is needed. Prevents speculative calls.'),
})

export const searchWebTool = (
	braveApiKey: string,
	sessionCallCounter: { count: number },
) => tool({
	description: 'Search the web for current, factual information. Use when the answer cannot come from memory or session context. Rate-limited to 5 calls per session.',
	inputSchema: searchWebSchema,
	execute: async (input) => {
		if (sessionCallCounter.count >= SESSION_WEB_SEARCH_LIMIT) {
			return {
				error: 'session_web_search_limit_reached',
				limit: SESSION_WEB_SEARCH_LIMIT,
				hint: 'Web search limit reached for this session. Use memory or ask the user to search themselves.',
			}
		}
		sessionCallCounter.count += 1

		const url = new URL('https://api.search.brave.com/res/v1/web/search')
		url.searchParams.set('q', input.query)
		url.searchParams.set('count', '5')

		const res = await fetch(url.toString(), {
			headers: {
				'Accept': 'application/json',
				'Accept-Encoding': 'gzip',
				'X-Subscription-Token': braveApiKey,
			},
		})

		if (!res.ok) {
			return { error: 'search_api_error', status: res.status, hint: 'Brave Search API returned an error. Try rephrasing the query.' }
		}

		const data = await res.json() as {
			web?: {
				results?: Array<{
					title: string
					url: string
					description: string
				}>
			}
		}

		const results = data.web?.results ?? []
		return {
			query: input.query,
			call_count: sessionCallCounter.count,
			remaining_calls: SESSION_WEB_SEARCH_LIMIT - sessionCallCounter.count,
			results: results.map(r => ({
				title: r.title,
				url: r.url,
				snippet: r.description,
			})),
		}
	},
})
```
