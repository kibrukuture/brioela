# 0001 — BrioelaIdentity Constant

## Scope

Brain / Agent identity

## What

Create `identity-prompt.ts` — one file, one exported constant: `BrioelaIdentity`.

800-token hard cap. Universal — same string for every user, every session. Deployed with the Worker, not stored in SQLite. Imported by `build.system.prompt.handler.ts` as Block 1 of every system prompt.

## Verification

- File exists at `backend/src/agents/brain/identity-prompt.ts`
- Exports `BrioelaIdentity` as a `const string`
- Token count ≤ 800 (estimate: ~4 chars/token)
- No per-user branching, no dynamic content
