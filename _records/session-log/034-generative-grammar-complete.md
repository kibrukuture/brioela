## Product Direction Captured

- Name: Brioela Generative Grammar.
- Do not generate runtime JSX/TSX.
- Do not use runtime MDX for core app UI.
- Do not build 1000 rigid full components as the creative strategy.
- AI generates typed grammar documents.
- Grammar primitives are like letters; composition creates emotional range.
- React Native renders compiled primitives only.
- PWA/web can use the same grammar payload.
- CopilotKit/AG-UI/A2UI are inspiration, not the core dependency decision.
- Static safety UI always wins.
- Skia, Reanimated, and haptics are selected through approved tokens only.
- Build-time AI can propose new components, but humans review and ship before runtime use.
